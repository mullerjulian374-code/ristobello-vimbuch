# Ristobello im Clubhaus Vimbuch

Website der Pizzeria **Ristobello im Clubhaus SV Vimbuch** — authentische italienische
Küche im Herzen des Schwarzwalds.

## Seiten
- `index.html` — Startseite mit immersivem Scroll-Erlebnis
- `speisekarte.html` — Speisekarte (mit Anker-Links pro Kategorie)
- `ueber-uns.html` — Über uns / Geschichte
- `galerie.html` — Galerie
- `kontakt.html` — Kontakt, Öffnungszeiten & Impressum

## Technik
Statische Website (HTML/CSS/JS). Animationen mit [GSAP](https://gsap.com/) +
ScrollTrigger und sanftem Scrollen via [Lenis](https://github.com/darkroomengineering/lenis).

## Lokal ansehen
```bash
python3 serve.py
# dann http://localhost:8000 öffnen
```
